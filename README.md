# Authortie

Был разработан сайт сервиса взаимопомощи между учёными Authortie.

Сайт написан на React с использованием TypeScript и собирается с помощью Vite. Для управления состоянием используется Redux Toolkit.

## Структура сайта

Сайт представляет собой лендинг, состоящий из следующих секций:

- **Шапка сайта** с навигацией по разделам (Аналитика, Сообщество, Для организаций), кнопками входа и создания заявки, а также адаптивным бургер-меню для мобильных устройств.
- **Hero-секция** с основным описанием сервиса и призывом к действию.
- **«Как это работает»** — пошаговое объяснение процесса работы с платформой.
- **Сообщество** — список карточек учёных с категориями, ключевыми словами, описанием и вознаграждением. Поддерживает постепенное раскрытие (показать все / скрыть).
- **Подвал** с навигационными ссылками и иконками социальных сетей.

Модальное окно с формами **входа** и **регистрации** открывается через Redux-состояние, поддерживает закрытие по клику на оверлей и по нажатию Escape. Рендеринг модалки осуществляется через React Portal.
